# 摄入路径详细处理步骤

> 由 `ingest` 和 `compile` skill 共享引用。定义三条摄入路径的判断逻辑和详细处理流程。

## 路径判断

读取用户提供的内容，按此表判断：

| 输入特征 | 路径 | 可信度标记 |
|---------|------|--------|
| 人工写的读书笔记/摘录/书评/实验记录 | **路径 A：人工笔记→结构化** | `source: human-notes` |
| 论文/图书原文（MD/文本），用户说"帮我读" | **路径 B：LLM 直接阅读** | `source: llm-reading` |
| 人工笔记 + 原文都有，用户说"对照" | **路径 C：交叉验证** | `source: cross-validated` |

## 路径 A：人工笔记 → 结构化（优先路径）

人工笔记可以是任意格式——想到什么写什么。LLM 的任务是结构化，**不做推断补充**。人工的洞察不会被修改或替代。

### 论文笔记 → 论文知识卡

从笔记中提取：
- 元数据（如果有 BibTeX 或标题/作者线索）
- 方法评注 → `fulltext.md` 的方法章节
- 摘录的关键公式 → 数学表述
- 批判性评论 → 保留在"讨论"章节（人类独有价值的洞察）
- 与其他论文的比较 → 交叉链接线索

### 图书笔记 → 图书知识卡

- **必须保留人工标注的页码**（反幻觉的关键：页码是人亲眼看到的）
- 区分"摘要"和"书评"——人的批判性判断独立成节
- 多文件合并（如 ch1-notes.md + ch2-notes.md → 一张图书卡）

### 笔记 → 信息知识卡

- 尽量保留原始内容，LLM 只负责格式化和分类
- 标注信息缺失：笔记中没覆盖的章节标注 `<!-- TODO: 需补充 -->`

## 路径 B：LLM 直接阅读原文（无人笔记时的替代方案）

### 核心原则

1. **诚实标注**：生成的知识卡 frontmatter 中标记 `source: llm-reading` 和 `trust: review-needed`
2. **区分事实和推断**：原文明确说的 → 标注 `[原文]`；LLM 推断的 → 标注 `[LLM推断]`
3. **提醒用户**：开头声明"以下内容由 LLM 直接从原文生成，未经人工验证。建议在引用前对照原文确认。"

### 生成后提示用户

> ⚠️ 此知识卡由 LLM 直接阅读原文生成（`source: llm-reading`）。建议：
> 1. 快速浏览确认关键信息准确
> 2. 补充你个人的批判性评论
> 3. 如果之后写了人工笔记，重新编译可升级为 `cross-validated`

## 路径 C：交叉验证（人工笔记 + LLM 阅读）

这是**最高质量**的路径。LLM 同时阅读人工笔记和原文。

### 验证流程

1. **以人工笔记为 ground truth**：人的洞察优先
2. **逐章节对照**：
   - 摘要/引言/方法/实验/结论 —— 人工笔记有吗？与原文一致吗？
   - 一致的地方标记 `✅ cross-validated`
   - 人工笔记独有观点（原文没有的批判性评论）→ 保留，标记 `[人工洞察]`
   - 原文有但人工笔记未涉及 → LLM 补充，标记 `[LLM补充-请确认]`
3. **关键数据点核实**：实验数字、公式、作者姓名——人工笔记和原文一致吗？
4. **标记可信度**：
   - 核心部分一致 → `source: cross-validated`, `trust: high`
   - 有少量不一致 → `source: cross-validated`, `trust: review-needed`，附差异清单

### 交叉验证报告格式

嵌入到 `fulltext.md` 或 `book.md` 末尾：

```markdown
## 🔍 交叉验证记录

- 验证时间：YYYY-MM-DD
- 人工笔记：`raw/papers/<citekey>-notes.md`
- 原文：`raw/papers/<citekey>-fulltext.md`

### 一致性检查

| 章节 | 状态 | 说明 |
|------|------|------|
| 摘要 | ✅/⚠/📝 | ... |
| 方法 | ✅/⚠/📝 | ... |
| 实验 | ✅/⚠/📝 | ... |
| 结论 | ✅/⚠/📝 | ... |

### 人工独有洞察（保留）
- ...

### LLM 补充（请确认）
- [LLM补充] ...
```

## 生成后报告

向用户报告：

```markdown
- **摄入路径**：A / B / C
- **可信度标记**：source: xxx, trust: xxx
- **信息缺失项**：[如有]
- **提醒**：[如 source: llm-reading，提醒人工复审]
```
